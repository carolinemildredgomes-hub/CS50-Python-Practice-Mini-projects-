# 🎬 Movie Search API System

A CS50-style Python project designed to strengthen problem-solving, API usage, and real-world project development skills.

This project is based on **Python APIs, Requests Library, JSON, Loops, and Search Logic**.

It simulates a simple **movie search environment** where users can search movie titles using an API.

---

## 🎯 Project Objective

The main objective of this project is to practice core Python concepts from the course module in a practical and beginner-friendly way.

This project focuses on:

* API integration
* JSON handling
* CS50-style problem solving
* reusable search logic
* user input and loops

---

## 📌 Features

* 🎬 Search movies by title
* 🌐 API-based search
* 💻 Terminal interaction
* 📚 Loop-based result display
* ⚠️ Handles no-result cases

---

## 🛠 Concepts Used

### 📚 Libraries

* `requests`

### 🧠 Core Concepts

* APIs
* JSON
* loops
* conditions
* dictionaries
* user input
