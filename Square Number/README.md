# Square a Number

## Description
This is a beginner Python program that asks the user to enter a number and then calculates the square of that number.

The project was built while learning Python fundamentals such as variables, user input, and mathematical operations.

## Features
- Takes numeric input from the user
- Calculates the square of the number
- Displays the result in the terminal

## Technologies Used
- Python 3

## How It Works
1. The program asks the user to enter a number.
2. The number is stored in a variable.
3. The program multiplies the number by itself.
4. The result is printed.

## Example Output

Enter a number:  
5

Square:  
25

## How To Run

Run the following command in the terminal:

python square.py
