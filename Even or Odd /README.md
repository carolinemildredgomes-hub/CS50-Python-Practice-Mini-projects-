# Even or Odd Checker

## Description
This Python program checks whether a number entered by the user is even or odd.

## Features
- Uses conditional statements
- Uses modulo operator

## Technologies Used
Python 3

## How It Works

The program checks if the number divided by 2 leaves a remainder.

If remainder = 0 → Even  
Otherwise → Odd

## Example Output

Enter number:
8

Even

## How To Run

python even_odd.py
