# 🎮 Quiz Game

## 📌 Description
This program is a simple quiz game. Users answer multiple-choice or numeric questions, and their score is calculated.

---

## 🚀 Features
- Predefined questions and answers
- Handles wrong answers
- Shows final score

---

## 🧠 Concepts Used
- Dictionaries for questions
- Loops
- Exception handling
- Conditional logic

---

## ⚙️ How It Works
1. Loop through each question.
2. Input answer from user.
3. Compare answer to correct one.
4. Raise ValueError if wrong (optional, for message)
5. Count correct answers.
6. Display final score.

---

## ▶️ Example
2+2: 4
3+5: 9
Wrong answer.
10-2: 8
Final Score: 2
