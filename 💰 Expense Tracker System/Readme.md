# 💰 Expense Tracker System

A beginner-friendly Python project to manage daily expenses and practice core programming concepts.

---

## 📌 Features

- Add expense category and amount
- View all expenses
- Calculate total spending
- Save records to CSV file
- Menu-driven CLI application

---

## 🛠 Technologies Used

- Python
- CSV
- Functions
- Lists
- Dictionaries

---

## ▶️ How to Run

```bash
python expense_tracker.py
```

---

## 🎯 Learning Outcome

- file handling
- dictionaries
- loops
- functions
- calculations
