# 📚 Student Attendance System

A CS50-style Python project designed to strengthen problem-solving, function building, and real-world project development skills.

This project is based on **Python Functions, Dictionaries, User Input, Search Logic, and Custom Function Design**.

It simulates a simple **student attendance management environment** where users can check attendance status by entering a student’s name.

---

## 🎯 Project Objective

The main objective of this project is to practice core Python concepts from the course module in a practical and beginner-friendly way.

This project focuses on:

* Python functions
* dictionary search logic
* CS50-style problem solving
* reusable code structure
* input and output handling
* beginner-friendly program design

---

## 📌 Features

* 🎓 Search attendance by student name
* ✅ Show Present / Absent / Late status
* ⚠️ Handles unknown student names
* 💻 Easy terminal-based interaction
* 📚 Function-based search system
