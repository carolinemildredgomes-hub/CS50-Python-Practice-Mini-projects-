# 💸 Expense Tracker

## 📌 Description
This program allows users to track expenses. Users can enter expense names and amounts, which are stored and totaled at the end.

---

## 🚀 Features
- Input expense name and amount
- Handles invalid input (non-numeric or negative)
- Allows multiple entries
- Calculates total expenses
- Displays all expenses

---

## 🧠 Concepts Used
- Dictionaries for storing expenses
- Loops and input
- Exception handling
- Conditional logic
- Summing dictionary values

---

## ⚙️ How It Works
1. User inputs expense name and amount repeatedly.
2. Negative or invalid amounts raise ValueError.
3. Expenses stored in dictionary.
4. Input 'done' ends loop.
5. Display individual expenses and total.

---

## ▶️ Example


Expense name: food
Amount: 50
Expense name: transport
Amount: 20
Expense name: done

Expenses:
food: 50
transport: 20
Total: 7
