# Greeting Program

## Description
This is a simple Python program that asks the user for their name and prints a greeting message.

This project was created while learning Python fundamentals from Harvard's CS50 Python course.

## Features
- Takes user input
- Uses Python variables
- Prints formatted output

## Technologies Used
Python 3

## How It Works
1. The program asks the user to enter their name.
2. The name is stored in a variable.
3. The program prints a greeting message using the name.

## Example Output

What is your name?
Caroline

Hello, Caroline

## How To Run

1. Download the project
2. Open terminal
3. Run:

python greeting.py
