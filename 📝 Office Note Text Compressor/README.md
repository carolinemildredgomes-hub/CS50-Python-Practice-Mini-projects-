# 📝 Office Note Text Compressor

A CS50-style Python project designed to strengthen problem-solving, string manipulation, and unit testing skills.

This project is based on **Python Functions, String Methods, and Pytest Unit Testing**.

It simulates an **office note compressor** where all vowels are removed to create shorter text.

---

## 🎯 Project Objective

The main objective of this project is to practice Python functions, loops, and string processing in a beginner-friendly way.

This project focuses on:

* functions
* loops
* string methods
* unit testing
* CS50-style problem solving

---

## 📌 Features

* 📝 Removes vowels from notes
* 💻 Terminal-based interaction
* ✨ Supports mixed case text
* 🧪 Includes pytest unit tests
* 🧩 Modular function design
