# Last Letter of Word

## Description
This Python program asks the user to enter a word and prints the last letter of that word.

This project demonstrates basic string manipulation in Python.

## Features
- Accepts text input from the user
- Extracts the last character of a word
- Displays the result

## Technologies Used
- Python 3

## How It Works
1. The program asks the user to enter a word.
2. The word is stored in a variable.
3. The program accesses the last character using string indexing.
4. The last letter is printed.

## Example Output

Enter a word:  
Python

Last letter:  
n

## How To Run

Run the program using:

python last_letter.py
