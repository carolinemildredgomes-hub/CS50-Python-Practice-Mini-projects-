# 🥤 Factory Drink Tank Liquid Meter

A CS50-style Python project designed to strengthen problem-solving, exception handling, and unit testing skills.

This project is based on **Python Functions, Fractions, Exceptions, and Pytest Unit Testing**.

It simulates a **factory liquid tank percentage meter**.

---

## 🎯 Project Objective

This project focuses on:

* fractions
* percentage calculation
* exception handling
* unit testing
* CS50-style problem solving

---

## 📌 Features

* 🥤 Converts liquid fractions into percentages
* 💻 Terminal-based interaction
* ✨ Tank level display
* 🧪 Includes pytest unit tests
