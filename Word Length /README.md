# Word Length Counter

## Description
This Python program counts the number of characters in a word entered by the user.

The project demonstrates basic string handling and the use of Python's built-in functions.

## Features
- Takes word input from the user
- Calculates the number of characters
- Displays the word length

## Technologies Used
- Python 3

## How It Works
1. The user enters a word.
2. The program counts the number of characters in the word.
3. The total number of characters is printed.

## Example Output

Enter a word:  
Python

Word length:  
6

## How To Run

Run the program with:

python word_length.py
