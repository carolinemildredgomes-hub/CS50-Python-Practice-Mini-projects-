# 🐦 Bird Message ASCII System

A CS50-style Python project designed to strengthen problem-solving, function design, and creative terminal project development skills.

This project is based on **Python Functions, User Input, ASCII Art, and Output Formatting**.

It simulates a fun **bird messenger system** where users can type a custom message and display it with an ASCII bird shape.

---

## 🎯 Project Objective

The main objective of this project is to practice Python functions and creative terminal output in a practical and beginner-friendly way.

This project focuses on:

* function building
* user input
* output formatting
* CS50-style problem solving
* creative terminal design

---

## 📌 Features

* 🐦 Bird ASCII art
* 💬 Custom message display
* 💻 Terminal interaction
* 📚 Function-based design
