# 🎲 Dice Roller Simulator

A CS50-style Python project designed to strengthen problem-solving, random number generation, and real-world project development skills.

This project is based on **Python Random Library, Functions, Loops, and User Input**.

It simulates a **dice roller simulator** in the terminal where users can roll a dice multiple times.

---

## 🎯 Project Objective

The main objective of this project is to practice Python’s random module, loops, and function design in a beginner-friendly way.

This project focuses on:

* random number generation
* functions
* loops
* CS50-style problem solving
* user interaction

---

## 📌 Features

* 🎲 Roll a dice randomly
* 🔄 Roll multiple times
* 💻 Terminal-based interaction
* ⚠️ Handles invalid input
* 🧩 Function-based design

---

## 🛠 Concepts Used

### 📚 Libraries

* `random`

### 🧠 Core Concepts

* functions
* loops
* conditions
* user input
* random number generation
