# 🔢 Safe Integer Input

## 📌 Description
This Python program repeatedly asks the user for a number until a valid integer is entered. It prevents program crashes by handling invalid inputs gracefully.

---

## 🚀 Features
- Accepts only integers
- Handles invalid input using exceptions
- Loops until a valid number is provided
- Displays the valid integer once entered

---

## 🧠 Concepts Used
- Exception handling (`try` / `except`)
- Loops (`while True`)
- Input validation
- Conditional logic

---

## ⚙️ How It Works
1. The program asks the user to enter a number.
2. If the input cannot be converted to an integer, a `ValueError` occurs.
3. The exception is caught, and the user is prompted again.
4. Once a valid integer is entered, it is displayed and the program exits.

---

## ▶️ Example
Number: abc
Invalid input, please enter an integer.

Number: 25
Valid integer: 25
