# 🆔 Unique Username Generator

A CS50-style Python project designed to strengthen problem-solving, string manipulation, slicing, and real-world project development skills.

This project is based on **Python Strings, Lists, Slicing, Functions, and User Input**.

It simulates a **unique username generator** using slicing from the user’s full name.

---

## 🎯 Project Objective

The main objective of this project is to practice Python slicing, list manipulation, and function design in a beginner-friendly way.

This project focuses on:

* string slicing
* list handling
* reusable functions
* CS50-style problem solving
* user input handling

---

## 📌 Features

* ✨ Generates unique usernames from first and last name
* 🔡 Handles both short and long names
* 💻 Terminal-based interaction
* 🧩 Function-based design
