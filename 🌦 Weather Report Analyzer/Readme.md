# 🌦 Weather Report Analyzer

A Python project that fetches live weather data using an API.

---

## 📌 Features

- fetch live weather
- temperature
- humidity
- weather description

---

## 🛠 Technologies Used

- Python
- requests
- API
- JSON

---

## 📦 Installation

```bash
pip install requests
```

---

## ▶️ How to Run

```bash
python weather_report.py
```

## 🎯 Learning Outcome

- API integration
- JSON parsing
- real-time data
