# 📊 Grade Calculator

## 📌 Description
This Python program calculates a student's average score and assigns a grade. It takes multiple score inputs until the user stops entering values.

---

## 🚀 Features
- Accepts multiple scores
- Handles invalid inputs using exceptions
- Calculates average
- Assigns grade (A-F)
- Ends input with blank entry

---

## 🧠 Concepts Used
- Exception handling (`try` / `except`)
- Loops (`while True`)
- Lists to store scores
- Conditional grading

---

## ⚙️ How It Works
1. User enters scores repeatedly.
2. Invalid inputs are handled via `ValueError`.
3. Scores are stored in a list.
4. Average is calculated and grade assigned:
   - A: 90+
   - B: 80-89
   - C: 70-79
   - D: 60-69
   - F: <60

---
