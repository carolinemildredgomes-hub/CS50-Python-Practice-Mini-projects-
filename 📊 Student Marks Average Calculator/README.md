# 📊 Student Marks Average Calculator

A CS50-style Python project designed to strengthen problem-solving, statistics module usage, and real-world project development skills.

This project is based on **Python Statistics Library, Lists, User Input, and Built-in Functions**.

It simulates a **classroom marks management system** where users can input student marks and calculate statistics.

---

## 🎯 Project Objective

The main objective of this project is to practice Python’s statistics module, list handling, and basic data analysis in a beginner-friendly way.

This project focuses on:

* Python built-in libraries
* statistics calculation
* lists and loops
* user input handling
* CS50-style problem solving

---

## 📌 Features

* 📈 Calculate average marks
* 🏆 Find highest marks
* 🔽 Find lowest marks
* 💻 Terminal-based interaction
* 🧩 List-based input handling

---

## 🛠 Concepts Used

### 📚 Libraries

### 🧠 Core Concepts
