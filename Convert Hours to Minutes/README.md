# Hours to Minutes Converter

## Description
This Python program converts hours into minutes using a simple mathematical calculation.

It demonstrates how to use variables, arithmetic operations, and user input in Python.

## Features
- Accepts numeric input
- Performs a unit conversion
- Displays the result clearly

## Technologies Used
- Python 3

## How It Works
1. The program asks the user to enter hours.
2. The value is multiplied by 60.
3. The result is printed as minutes.

## Example Output

Enter hours:  
2

Minutes:  
120

## How To Run

Run the program with:

python hours_to_minutes.py
