# Temperature Converter

## Description
A Python program that converts temperature from Celsius to Fahrenheit using a mathematical formula.

## Features
- User input
- Mathematical calculation
- Output display

## Technologies Used
Python 3

## Formula Used

F = (C × 9/5) + 32

## Example Output

Enter Celsius temperature:
25

Fahrenheit: 77

## How To Run

python temperature.py
