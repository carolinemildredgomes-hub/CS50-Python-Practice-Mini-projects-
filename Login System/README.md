# 🔐 Login System

## 📌 Description
This Python program simulates a login system. Users have 3 attempts to enter the correct username and password.

---

## 🚀 Features
- Predefined username and password
- 3 login attempts
- Error messages for invalid credentials
- Account locks after 3 failed attempts

---

## 🧠 Concepts Used
- Loops and counters
- Exception handling
- Conditional logic

---

## ⚙️ How It Works
1. User enters username and password.
2. Invalid credentials raise ValueError.
3. Counter keeps track of attempts.
4. Success → login message
5. 3 failed attempts → account locked

---

## ▶️ Example

Username: admin
Password: 123
Error: Invalid credentials.

Username: admin
Password: 1234
Login successful!
