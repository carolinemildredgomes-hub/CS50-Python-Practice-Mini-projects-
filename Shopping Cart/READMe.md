# 🛒 Shopping Cart

## 📌 Description
This Python program simulates a shopping cart. Users can add items from a menu, and the program keeps track of quantities and total price.

---

## 🚀 Features
- Predefined menu with prices
- Case-insensitive input
- Validates item availability
- Updates quantity and total price
- Handles invalid items using exceptions
- Ends input by typing 'done'

---

## 🧠 Concepts Used
- Dictionaries for menu and cart
- Loops and input handling
- Exception handling
- Conditional statements

---

## ⚙️ How It Works
1. Menu stored as a dictionary.
2. User inputs items repeatedly.
3. Input converted to lowercase to avoid case mismatch.
4. If item exists, it’s added to cart and total updated.
5. Invalid items raise ValueError.
6. Input 'done' ends the loop.

---

## ▶️ Example

Item: apple
Added apple. Total = 2

Item: banana
Added banana. Total = 3

Item: water
Error: Item not available.

Item: done
